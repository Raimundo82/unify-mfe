### Usage

```sh
cd shell
yarn dev
```

From the root:

```sh
npm install -g ./create-unify-mfe
```

```sh
npx create-unify-mfe -name <some-name> -port <some-port>
```

```sh
cd <some-name>
yarn
yarn dev // dev server
yarn publish-dev // publish on the server
```

Go to http://localhost:`<some-port>/<some-name>`
