1. Clone the repo

```sh
    cd shell
   yarn build
```

2. From the root:

```sh
npm install -g ../
```

```sh
npx create-unify-mfe -name <some-name>
```

```sh
cd <some-name>
yarn dev // dev server
yarn publish-dev // publish on the server
```

3. Go to http://localhost:9000/ldod-mfes/<some-name>
