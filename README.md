### Usage

```sh
cd shell
yarn build
```

From the root:

```sh
npm install -g ../
```

```sh
npx create-unify-mfe -name <some-name>
```

```sh
cd <some-name>
yarn dev // dev server
yarn publish-dev // publish on the server
```

Go to http://localhost:9000/ldod-mfes/`<some-name>`

Note: Is not tottaly decoupled from the LdoD project but soon it will...
